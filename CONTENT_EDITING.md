# How to Edit Content

## Option 1: Prose.io (Recommended)

1. Go to: https://prose.io/#jtalborough/roosterbots-content
2. Click "Authorize on GitHub" (one-time)
3. Navigate to `content/` folder
4. Click any `.md` file to edit
5. Click the save icon when done

## Option 2: GitHub.com Direct

1. Go to: https://github.com/jtalborough/roosterbots-content
2. Navigate to `content/` folder  
3. Click any `.md` file
4. Click the pencil icon to edit
5. Click "Commit changes" when done

## Option 3: GitHub.dev (VS Code in Browser)

1. Go to: https://github.com/jtalborough/roosterbots-content
2. Press `.` key on your keyboard
3. Full VS Code opens in browser
4. Edit files, then use Source Control to commit

## Content Structure

- `content/blog/` - Blog posts
- `content/about.md` - About page
- `content/donate.md` - Donate page
- `content/store/` - Store content

Changes will be live after the site rebuilds (usually 2-3 minutes).
